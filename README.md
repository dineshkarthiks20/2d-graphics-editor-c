# 2D Graphics Editor

## Description
A console-based 2D Graphics Editor implemented in C++.

## Features

- Draw Rectangle
- Draw Circle
- Draw Line
- Draw Triangle
- Add Shapes
- Delete Shapes
- Modify Shapes
- Display Canvas

## Data Structure

A 2D character array is used as the drawing canvas.

```cpp
char canvas[30][60];
```

## Compilation

```bash
g++ main.cpp -o editor
./editor
```
